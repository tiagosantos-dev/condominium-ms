# Technologies used
## Back end
- Java
- Spring Boot
- Spring Cloud
- JPA / Hibernate
- Maven
- Liquibase

## Deployment into production
- Database: Postgresql

## Deployment in development
- Database: H2

# How to run the project

## Backend
Requirements: Java 11

```bash
# clone repository
git clone https://github.com/tiagosantos-dev/condominium-ms
# enter back end project folder
cd condominium-ms
# run the project
./mvnw spring-boot:run
```
